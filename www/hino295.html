295- Nda o yongola esumuluho


1.	
	Nda o yongola esumuluho
	Kuama volomai via Yesu.
	O ku muisa esanju lia lua;
	Kuama volomai via Yesu.

		Linga ondonge yaíe;
		U kuama lekolelo.
		Likakatela laíe.
		Kuama volomai via Yesu.
2.	
	Nda o yongola epuyuko
	Kuama volomai via Yesu.
	Nda wa kava o ku kolisa.
	Kuama volomai via Yesu.

3.	
	Nda o yongola ekolowa
	Kuama volomais via Yesu.
	Ka tikule omindikiso;
	Kuama volomai via Yesu.

4.	
	Nda o yongola viokovailu
	Kuama volomai via Yesu.
	Eye lika Ongavi yaco.
	Kuama volomai via Yesu.
